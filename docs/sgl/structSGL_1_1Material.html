Detailed Description

Represents a material with properties for rendering.

The Material struct contains properties such as color, ambient, diffuse, specular, and shininess, which are commonly used in lighting calculations for 3D rendering. It also provides methods for JSON serialization and deserialization, as well as utility functions to set the material's color.
